# SmartWatering System

![SmartWatering System](../images/smart-watering.jpg)

## Intelligent bevattning som sparar upp till 60% vatten

SmartWatering System är ett smart bevattningssystem som automatiskt anpassar vattningen efter väder, jordtyp och växternas behov.

### Funktioner

- **Väderanpassad:** Integreras med lokala väderdata
- **App-styrning:** Kontrollera från din smartphone
- **Zonindelning:** Upp till 12 olika zoner med individuella inställningar
- **Läckagedetektering:** Varnar automatiskt vid läckage
- **Vattenrapporter:** Se exakt hur mycket vatten du sparar

### Tekniska specifikationer

- Strömförsörjning: 230V eller solcell
- Vattenflöde: 1-15 liter/minut per zon
- Trådlös anslutning: WiFi 2.4GHz
- App: iOS och Android
- Garanti: 2 år
