# EcoGarden Solutions - Företagshemsida

Detta är en komplett mappstruktur med markdown-filer för en företagshemsida.

## Om företaget

**EcoGarden Solutions** är ett fiktivt företag som säljer hållbara trädgårdsprodukter:
- SmartWatering System (intelligent bevattning)
- CompostPro 3000 (automatisk kompostering)
- SolarGrow Lights (solcellsdrivna växthusbelysning)

## Mappstruktur

```
company-website/
├── index.md                    # Startsida
├── products.md                 # Produktöversikt
├── shop.md                     # Webbshop
├── contact.md                  # Kontakt

│
├── products/                   # Produktsidor
│   ├── smart-watering.md
│   ├── compost-pro.md
│   └── solar-grow.md
│
├── about/                      # Om företaget
│   ├── index.md                # Om oss

│
├── support/                    # Support & hjälp

```

### Som CMS-content
Importera markdown-filerna till ditt CMS (WordPress, Drupal, etc).

### Som utvecklingsmall

### Som dokumentation

## Tekniska detaljer

- **Format:** Markdown (.md)
- **Kodning:** UTF-8
- **Språk:** Svenska
- **Länkar:** Relativa länkar mellan filer
- **Bilder:** Placeholders (behöver ersättas med riktiga bilder)

## Anpassning

För att anpassa till ditt företag:

1. **Globalt sök och ersätt:**
   - "EcoGarden Solutions" → Ditt företagsnamn
   - "Trädgårdsgatan 12" → Din adress
   - "08-123 456 78" → Ditt telefonnummer
   - "info@ecogarden.se" → Din e-post

2. **Byt produkter:**
   - Redigera produktsidorna i `/products/`
   - Uppdatera produktöversikten i `products.md`

3. **Juridiska dokument:**
   - Uppdatera organisationsnummer
   - Anpassa köpvillkor efter din verksamhet
   - Konsultera jurist för GDPR-compliance

4. **Lägg till bilder:**
   - Ersätt image-placeholders med riktiga bildlänkar
   - Skapa mapp `/images/` och lägg till bilder
